Part 1. Human Cognitive Architecture (Asymmetric Memory Matrix)

Rather than treating learning loss as a flat, uniform 50% drop-off across all parameters, knowledge recall is modeled using a non-linear, three-tiered probability profile based on task complexity and instruction isolation[cite: 5, 6, 7]:

Cognitive Tier Recall Probability Evaluation Baseline Criteria
Tier 1: Highlighted Training Material [cite: 10] 90% – 95% [cite: 10] Explicit "Symptom-to-Resolution" data pairs or parameters isolated for RAG fine-tuning[cite: 11]. Near-perfect memory recall; resolution occurs without substantial structural lag[cite: 13].
Tier 2: Textbook Basics [cite: 14] 75% – 80% [cite: 14] High-level chapter headings and operational loops (e.g., OS screen duplication commands, basic cable ID, power continuity checks)[cite: 15, 18, 19]. Retained conceptually but introduces minor operational execution friction[cite: 20].

Part 2. Localized Resource Envelope

The simulation strictly limits outcomes to the local classroom environment. No external tech resources or swap-outs exist outside this exact boundary[cite: 30, 31, 32]:

  • Fixed Architecture: Classrooms possess exactly one alternate auxiliary routing pathway (e.g., an available legacy VGA line if the HDMI line exhibits digital drop-out)[cite: 33].
  • Proactive Tech Kit: The educator carries an inventory consisting strictly of: 1x spare digital patch cable (HDMI/USB touch-tether), 1x USB-C to HDMI adapter dongle, 1x standard IEC power cord, and a minimal pack of AA/AAA dry-cell batteries[cite: 34, 36, 37, 38, 39].
  • Hard Architectural Boundaries: Total failure of main structural devices (e.g., blown projector lamps, shattered LCD panels, broken built-in podium microphone lines) cannot be physically bypassed or hot-swapped[cite: 40, 41].

Part 3. Continuity & Time-Recovery Rubric

Prioritizes instructional continuity and lesson preservation over engineering or technical aesthetics[cite: 44].

Score Metric Operational Threshold Pedagogical Impact Definition
Score 3 Diagnosed & bypassed within 120 seconds[cite: 48]. Full Resolution / Workaround: Lesson proceeds completely as planned; zero media modification or student disengagement[cite: 45, 49, 50].
Score 2 Functionality restored via alternative patch[cite: 54]. Diminished Capability: Device works, but restrictions compromise lesson format or force content omission (e.g., loss of a color channel)[cite: 52, 54, 55, 56].
Score 1 Hardware break prevents immediate recovery[cite: 59]. Minor Mitigation / Accurate Routing: Presentation abandoned, forcing legacy backup methods[cite: 58, 61]. The accurate diagnostic report reduces long-term maintenance latency[cite: 62].
Score 0 Persistent fault, complete lockout[cite: 65]. Total Abandonment: Teacher abandons environment completely due to technology fatigue and system mistrust[cite: 63, 66].
